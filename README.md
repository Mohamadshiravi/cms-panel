## Getting Started

First, build project:

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```

and , run project:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
